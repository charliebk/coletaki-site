# Coletaki — Site Institucional

Site estático da **Coletaki** — plataforma climate-tech de gestão de resíduos sólidos urbanos no Brasil. Otimizado para hospedagem gratuita em **GitHub Pages** com domínio personalizado.

## Estrutura do projeto

```
coletaki-site/
├── index.html              ← Landing page principal
├── assets/style.css        ← Estilização
├── mockups/                ← 4 mockups SVG
├── .nojekyll               ← Desativa Jekyll
├── CNAME                   ← Domínio personalizado
└── README.md
```

## Stack

- HTML5 + CSS3 + JavaScript vanilla
- Hospedagem: GitHub Pages
- Branding: Paleta Litoral + Cabinet Grotesk + Inter
- Responsive: Mobile-first
